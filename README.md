Login Ship

## Developing

Ensure that [Node.js](http://nodejs.org) and [gulp.js](http://gulpjs.com) are installed.

```
git clone git@github.com:hull-ships/hull-login.git
cd hull-login
npm install
gulp
```

## Grunt tasks

- Linting: `gulp lint`
- Building: `gulp build`
- Deploying: `gulp deploy`
